## \Features

### Advanced Search & Discovery
Search through NASA's entire exoplanet database with ease. Find specific exoplanets and explore comprehensive data including:
- Host star information
- Discovery method and year
- Orbital characteristics (radius, period)
- Physical properties (mass, radius, temperature)

### Interactive Data 
- Click any column header to sort exoplanets by that property
- Toggle between ascending and descending order
- 
### Rich Data Visualizations
Explore exoplanet data through six comprehensive graphs:
- Discovery trends over time
- Mass and radius distribution
- Orbital characteristics
- Temperature ranges
- Discovery method

## ðŸš€ Getting Started

### Prerequisites
- Node.js and npm installed
- Python 3.x installed

### Installation

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Update Exoplanet Database** 
   ```bash
   python exoplanet_etl.py
   ```

### Running the Application

1. **Start the Backend**
   ```bash
   node exoplanet_backend.js
   ```

2. **Launch the Frontend** (in a new terminal)
   ```bash
   npm start
   ```
